6 - GIT TAGS

git commit --amend -m "Cambio en h3" ... Modifica descripción del commit último
git push origin master -f ... fuerza a github a subir el cambio

git tag -a v0.8 -m "Version 0.8 de nuestro proyecto" ... tag anotadas, comando aplica al último commit
git log ... copia el código
git tag -a v0.8 -m "Version 0.1 de nuestro proyecto" <<copia codigo anterior>>
git tag v1.0 ... tag ligeras

subirlo a gitHub
git push origin v0.8 ... sube solamente una versión de tags
git push origin --tags ... se encarga de subir todos los tags
